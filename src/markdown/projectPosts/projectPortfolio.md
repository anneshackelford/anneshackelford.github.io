## Development Tools and Resources

### Hosting
* [Heroku Cloud Application Platform](https://www.heroku.com/) for hosting website, connection to database, deploying from GitHub 
* [GitHub](https://github.com/) for storing code 
* [Google Custom Domain](https://domains.google/) 

### Coding
* [Visual Studio Code](https://code.visualstudio.com/) as development platform
* [HackMD](https://hackmd.io/) as as online Markdown Editor which uses [Imgur](https://imgur.com/) to create links to images to be used in the markdown files
* [Squoosh](https://squoosh.app/) to shrink image sizes 
* [Unsplash](https://unsplash.com/) to find free images 
* [Picsum](https://picsum.photos/) for random images 
* [Carbon](https://carbon.now.sh/) for formatting screen captures 
* [Excalidraw](https://excalidraw.com/) to Sketch Hand-Drawn like diagrams 

### Managing
* [pgAdmin](https://www.pgadmin.org/) for PostgreSQL
